## My Setup

* **OS**: Arch Linux
* **Terminal**: Kitty
* **WM**: Hyprland
* **Bar**: Waybar
* **Font**: Hack Nerd Font
* **Launcher**: Wofi
* **Session Management**: Wlogout
* **Notifications**: Mako
