# Dotfiles

**My Hyprland desktop dotfiles with a custom theme switcher!**

</div>

## My Setup

* **OS**: Arch Linux
* **Terminal**: Kitty
* **WM**: Hyprland
* **Bar**: Waybar
* **Font**: Hack Nerd Font
* **Launcher**: Wofi
* **Session Management**: Wlogout
* **Notifications**: Mako
